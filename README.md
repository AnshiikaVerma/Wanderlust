# 🏡 Wanderlust

**Wanderlust** is a full-stack Airbnb-inspired web application that allows users to discover, create, and manage property listings with secure authentication, image uploads, reviews, and interactive maps.

🌐 **Live Demo:** https://majorproject-wmsz.onrender.com/listings

💻 **GitHub Repository:** https://github.com/AnshiikaVerma/Wanderlust

---

# 📖 Project Overview

Wanderlust is a full-stack rental platform inspired by Airbnb, built using the MERN ecosystem. The application allows users to explore property listings, create their own listings, upload images, write reviews, and securely manage their content.

The project follows the MVC architecture and implements authentication, authorization, server-side validation, cloud-based image storage, and interactive maps to provide a complete rental listing experience.
